## Examen tecnico de  Pigmalion Software (Hecho en JavaScript, node.js)
https://replit.com/@PabloCastro19/examen-tecnico-1

- Creamos una funcion 'encontrarSumaDosElementos' en donde se le pasa el array y el numero requerido.

- Dentro de la funcion creamos un objeto vacio llamado 'complementos'.

- Y hacemos un for para recorrer el array y obtener cada numero puesto en el array, que lo guardamos en la constante 'num'.

-  Creamos la constante 'complemento' para obtener el numero de la resta 'requiredSum' - 'num'.

- De esta manera si 'complemento' existe en el objeto 'complementos' podremos decir que se encontraron los dos numero que sumando dan el 'requiredSum', entonces el Output sera True.

- En caso de que no exista el 'complemento' en el objeto 'complementos', el 'num' sera agregado en el objeto 'complementos'.

- En caso de que haya finalizado la iteracion del array y no hubo coincidencia, el Output sera False.
